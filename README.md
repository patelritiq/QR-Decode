# QR Decode Project

## Description
The QR Decode project is a Python application that scans and decodes QR codes from images using OpenCV and PyZbar libraries. It provides a simple interface to read QR codes and extract the encoded data.

## Installation
To run this project, you need to have Python installed on your machine. You also need to install the required libraries. You can do this using pip:

```bash
pip install opencv-python pyzbar
```

## Usage
1. Place the image file containing the QR code in the same directory as the script or specify the path to the image.
2. Run the script:

```bash
python QR.py
```

3. The script will output the type and data of the QR code found in the image.
